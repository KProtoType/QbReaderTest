Quiz Bowl Practice App

Step-by-Step Installation Guide

1

Download the Code

First, you'll need to download the Quiz Bowl Practice App code to your computer.

[Image showing GitHub download button and ZIP download process]

Option A: Download ZIP File (Recommended for Beginners)

  1. Go to the GitHub repository
  2. Click the green "Code" button near the top of the page
  3. Select "Download ZIP" from the dropdown menu
  4. Save the ZIP file to your computer
  5. Extract the ZIP file to a location you can easily find (like your Documents folder)

Option B: Clone with Git (For GitHub Users)

If you're familiar with Git, you can clone the repository using this command:

git clone https://github.com/KProtoType/QbReaderTest.git

Tip

Make note of where you saved or extracted the files. You'll need to navigate to this location in the next step.

2

Install Node.js

The Quiz Bowl Practice App requires Node.js to run. If you don't already have it installed, you'll need to download and install it first.

[Image showing Node.js download page]

Download and Install Node.js

  1. Go to the Node.js website
  2. Download the "LTS" (Long Term Support) version for your operating system
  3. Run the installer and follow the installation prompts
  4. Accept the default settings if you're unsure

Verify Installation

To verify Node.js installed correctly, open a command prompt or terminal and type:

node --version

You should see a version number (e.g., v16.14.0) if the installation was successful.

Important!

You need Node.js version 14 or higher for the app to work properly.

3

Install Required Packages

Now you need to install the required packages that the app depends on.

[Image showing terminal with npm install command]

Open a Terminal/Command Prompt

  1. Open a terminal (Mac/Linux) or command prompt (Windows)
  2. Navigate to the folder where you extracted or cloned the Quiz Bowl files using the cd command:
cd path/to/QbReaderTest

Install Dependencies

Once you're in the correct folder, run the following command:

npm install

This will download and install all the required packages. It might take a minute or two to complete.

Tip: Finding Your Path

On Windows, you can copy the file path by navigating to the folder in File Explorer, clicking in the address bar, and copying the text.

On Mac, you can drag the folder onto the Terminal window to automatically insert its path.

4

Start the Server

Now you're ready to start the Quiz Bowl Practice App server!

[Image showing terminal with server running]

Start the Application

In the same terminal or command prompt window (make sure you're still in the QbReaderTest folder), run:

node server.js

You should see a message that says:

Quiz Bowl Practice Server running on port 5000

This means the server is running successfully!

Keep the Terminal Open

Important: Keep this terminal window open while you're using the app. If you close it, the app will stop running.

Troubleshooting

If you see an error message like "port 5000 is already in use", it means another program is already using that port. You can either:

  • Close the other program, or
  • Edit the server.js file to use a different port (look for "PORT" near the top of the file)
5

Open the Application

Now that the server is running, you can open the Quiz Bowl Practice App in your web browser.

[Image showing the app running in a web browser]

Access the App

  1. Open your web browser (Google Chrome is recommended for best speech recognition)
  2. Enter the following address in the address bar:
http://localhost:5000

The Quiz Bowl Practice App should now load in your browser!

Grant Microphone Permissions

When you start the game, your browser will ask for permission to use your microphone. You must allow this for the speech recognition to work.

Tip: Sharing with Others

To make the app available to others on your local network:

  1. Find your computer's IP address
  2. Others can access your app at http://YOUR_IP_ADDRESS:5000
